Alias Shortcuts

Aliases allow you to access websites quickly from the keyboard. To use the feature, just type the shortcut Alt + x to open the popup and then type the alias. Another way to use it is by typing x + space in the browser url bar.

How to set up your alias

Write it in the following format:
ALIAS space URL_AND_PARAMS. Where the parameters are written with the sign "|" + a number. Example:

git https://github.com/search?q=|1

The result of typing the alias "git" + the parameter "Owl Tools" will return a search on the GitHub website for the "Owl Tools".

Other tips

• You can put more than one parameter by enumerating them, example: |1 |2 |3

• It is also possible not to set any parameters.

• The character "*" get everything typed that is not an alias.
